# Stellar Legacy

Stellar Legacy is a generational starship strategy game about sending a living ship-civilization on missions that take decades or centuries to complete. You do not command a disposable crew. You build a legacy. Captains age out, children inherit roles, promises outlive the people who made them, and every voyage reshapes the society aboard your vessel.

## Core Fantasy

- Run a starship that is also a city, a company, and a dynasty.
- Take on impossible long-term contracts.
- Build trade power across generations.
- Leave behind a name that future captains must either honor or escape.

## Core Gameplay Pillars

- **Generational Command**: Leaders retire, die, and are replaced by heirs or successors.
- **Century-Scale Missions**: Contracts may take multiple generations to complete.
- **Shipboard Society Management**: Balance resources, morale, education, housing, and stability.
- **Trade Empire Growth**: Establish routes, monopolies, partnerships, and political influence.
- **Legacy and Consequence**: Each generation inherits wealth, debt, reputation, and old mistakes.

## Core Gameplay Loop

1. Accept a major long-term mission.
2. Prepare the ship, crew families, cargo, and finances.
3. Survive the journey through shortages, events, and internal politics.
4. Complete or abandon the mission decades later.
5. Expand the ship and empire with the results.
6. Hand the next era to a new generation of leadership.

## Current Prototype Focus

The current build explores resource management, market trading, ship systems, crew management, and legacy foundations. The long-term goal is to deepen the generational simulation so the ship feels like a civilization crossing centuries, not just a crew on a run.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Kalaith/stellar_legacy.git
   cd stellar_legacy/frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:5173`.
